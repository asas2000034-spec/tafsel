# منصة تفصيل (Tafseel Platform)

منصة متكاملة للتسوق والتصميم والمزادات مع نظام BNPL و NFT

## المميزات الرئيسية

### 🛍️ المول
- عرض المنتجات المخصصة
- نظام شراء متكامل
- دعم العملة الرقمية TFT

### 🎨 نظام التصميم
- رفع التصاميم وتوثيقها كـ NFT
- ربط التصاميم بالمصانع
- حساب الأسعار تلقائياً

### 🏭 نظام المصانع
- إدارة كتالوج المنتجات الخام
- استقبال الطلبات
- تنفيذ الطلبات وتتبعها

### 📢 نظام التسويق
- إنشاء روابط تتبع
- تتبع النقرات والمبيعات
- كسب العمولات

### 💳 نظام المحفظة
- شحن المحفظة
- نظام BNPL (70/30)
- رصيد مقفل ورصيد حر

## التقنيات المستخدمة

- **Next.js 14** - إطار عمل React
- **TypeScript** - للبرمجة الآمنة
- **Tailwind CSS** - للتصميم
- **React Icons** - للأيقونات
- **Zustand** - لإدارة الحالة (اختياري)

## التثبيت والتشغيل

### المتطلبات
- Node.js 18+ 
- npm أو yarn

### خطوات التثبيت

1. تثبيت الحزم:
```bash
npm install
```

2. تشغيل المشروع في وضع التطوير:
```bash
npm run dev
```

3. فتح المتصفح على:
```
http://localhost:3000
```

## البنية التحتية

### الصفحات الرئيسية

#### الرحلة العالمية
- `/` - الصفحة الرئيسية (Landing Page)
- `/register` - إنشاء حساب جديد
- `/login` - تسجيل الدخول
- `/role-selection` - اختيار الدور

#### رحلة العميل
- `/app/mall` - المول
- `/app/product/[id]` - صفحة المنتج
- `/app/checkout/[id]` - تأكيد الدفع
- `/app/wallet` - المحفظة
- `/app/orders` - الطلبات
- `/app/auction` - المزادات

#### رحلة المصمم
- `/designer/dashboard` - لوحة تحكم المصمم
- `/designer/upload` - رفع تصميم جديد
- `/designer/dual-choice` - اختيار طريقة البيع
- `/designer/store-setup` - إعداد المنتج للمول
- `/designer/success` - نجاح النشر

#### رحلة المصنع
- `/factory/dashboard` - لوحة تحكم المصنع
- `/factory/catalog` - إدارة الكتالوج
- `/factory/add-product` - إضافة منتج خام
- `/factory/order/[id]` - تنفيذ الطلب

#### رحلة المسوّق
- `/marketer/dashboard` - لوحة تحكم المسوّق
- `/marketer/new-campaign` - إنشاء حملة جديدة
- `/marketer/campaign/[id]` - تفاصيل الحملة

## نظام الأدوار

1. **عميل (Customer)** - للتسوق والشراء
2. **مصمم (Designer)** - لرفع التصاميم وبيعها
3. **مصنع (Factory)** - لتقديم خدمات الإنتاج
4. **مسوّق (Marketer)** - للترويج وكسب العمولات

## نظام BNPL

عند اختيار تمويل BNPL:
- 70% من المبلغ → رصيد حر
- 30% من المبلغ → رصيد مقفل (للاستخدام داخل المنصة فقط)
- عند الشراء: يتم الخصم من الرصيد المقفل أولاً

## ملاحظات التطوير

- البيانات حالياً محفوظة في `localStorage` (للتطوير فقط)
- في الإنتاج، يجب ربط التطبيق بخادم حقيقي و قاعدة بيانات
- نظام NFT يحتاج ربط حقيقي بـ Polygon Network
- نظام الدفع يحتاج ربط بـ TabaPay/Tamara APIs
