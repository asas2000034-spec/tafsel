# منصة تفصيل (Tafseel Platform)

منصة متكاملة تجمع بين التسوق والإبداع والاستثمار

## المميزات

- 🛍️ **المول**: تسوق منتجات متميزة
- 🎨 **NFT**: توثيق وحماية إبداعاتك على Polygon
- 💰 **كاش**: نظام محفظة مع تمويل BNPL (70/30)
- 📢 **ماركتنج**: نظام إحالات لكسب العمولات

## الأدوار والرحلات

### 1. رحلة العميل (Customer Journey)
**الشاشات:** 4-10

- تمويل المحفظة عبر BNPL (70% حر، 30% مقفل)
- التسوق من المول
- استخدام الرصيد المقفل أولاً عند الشراء
- تتبع الطلبات

**المسارات:**
- `/app/mall` - المول
- `/app/wallet` - المحفظة
- `/app/product/[id]` - صفحة المنتج
- `/app/checkout/[id]` - تأكيد الدفع
- `/app/orders` - طلباتي

### 2. رحلة المصمم (Designer Journey)
**الشاشات:** 11-15

- رفع التصاميم وتوثيقها كـ NFT على Polygon
- بيع التصاميم كمنتجات ملموسة في المول
- اختيار المصنع وتحديد السعر
- توزيع الأرباح (30% للمنصة، الباقي للمصمم)

**المسارات:**
- `/designer/dashboard` - لوحة التحكم
- `/designer/upload` - رفع تصميم جديد
- `/designer/setup/store` - إعداد المنتج للبيع

### 3. رحلة المصنع (Factory Journey)
**الشاشات:** 16-20

- إضافة منتجات خام للكتالوج
- استقبال الطلبات من العملاء
- تنفيذ الطلبات (طباعة، تغليف، شحن)
- توزيع الأرباح تلقائياً عند تأكيد الشحن

**المسارات:**
- `/factory/dashboard` - لوحة التحكم
- `/factory/catalog/add` - إضافة منتج خام
- `/factory/order/[id]` - تنفيذ الطلب

### 4. رحلة المسوّق (Marketer Journey)
**الشاشات:** 21-23

- إنشاء روابط تتبع للمنتجات
- كسب عمولات من المبيعات
- تتبع الأداء (النقرات، المبيعات، الأرباح)

**المسارات:**
- `/marketer/dashboard` - لوحة التحكم
- `/marketer/campaign/new` - إنشاء حملة جديدة
- `/marketer/campaign/[id]/success` - رابط التتبع

## التثبيت والتشغيل

```bash
# تثبيت المكتبات
npm install

# تشغيل المشروع في وضع التطوير
npm run dev

# بناء المشروع للإنتاج
npm run build

# تشغيل المشروع بعد البناء
npm start
```

المشروع سيعمل على `http://localhost:3000`

## الميزات الرئيسية

### نظام BNPL (70/30)
- عند التمويل عبر BNPL، يتم تقسيم الرصيد:
  - 70% رصيد حر (يمكن سحبه)
  - 30% رصيد مقفل (للاستخدام داخل المنصة فقط)
- عند الشراء، يتم الخصم من الرصيد المقفل أولاً

### نظام NFT
- توثيق التصاميم كـ NFT على Polygon
- حفظ حقوق الملكية
- إمكانية البيع كأصل رقمي

### نظام الإحالات
- إنشاء روابط تتبع فريدة
- تتبع النقرات والمبيعات
- حساب العمولات تلقائياً

## ملاحظات مهمة

⚠️ **البيانات الحالية:**
- البيانات محفوظة مؤقتاً في `localStorage`
- يحتاج المشروع إلى قاعدة بيانات حقيقية للإنتاج

🔧 **التكاملات المطلوبة:**
- نظام NFT: تكامل مع Polygon Blockchain
- نظام BNPL: تكامل مع تابي/تمارا
- نظام الدفع: تكامل مع بوابات الدفع (Mada, Apple Pay)
- قاعدة البيانات: MongoDB أو PostgreSQL
