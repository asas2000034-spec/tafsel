# منصة تفصيل (Tafseel Platform)

منصة متكاملة للتصاميم والمنتجات المخصصة مع نظام توثيق NFT وتمويل BNPL.

## 🌟 المميزات

### 1️⃣ رحلة العميل (Customer Journey)
- **المول الإلكتروني**: تصفح المنتجات المخصصة بتصاميم فريدة
- **نظام المحفظة (تفصيل كاش)**: 
  - رصيد حر (متاح للاستخدام في أي مكان)
  - رصيد مقفل (يُستخدم أولاً في المشتريات داخل المنصة)
- **تمويل BNPL**: شحن المحفظة بنظام 70/30 (0% فائدة)
- **نظام الشراء**: خصم تلقائي من الرصيد المقفل أولاً
- **تتبع الطلبات**: متابعة حالة جميع طلباتك

### 2️⃣ رحلة المصمم (Designer Journey)
- **رفع التصاميم**: واجهة سهلة لرفع التصاميم
- **توثيق NFT تلقائي**: حماية الحقوق الفكرية على Polygon
- **خيارات البيع المزدوجة**:
  - البيع كمنتج ملموس في المول
  - البيع كأصل رقمي في المزاد
- **إعداد المنتج**: اختيار المنتج الخام + المصنع + تحديد الربح
- **تتبع المبيعات**: إحصائيات شاملة عن التصاميم والأرباح

### 3️⃣ رحلة المصنع (Factory Journey)
- **كتالوج المنتجات**: إدارة المنتجات الخام المتاحة
- **إدارة الطلبات**: عرض وتنفيذ الطلبات الجديدة
- **تفاصيل الطلب**:
  - تحميل ملف التصميم
  - طباعة بوليصة الشحن
  - تأكيد الإنتاج والشحن
- **توزيع الأرباح**: تلقائي عند تأكيد الشحن

### 4️⃣ رحلة المسوق (Marketer Journey)
- **إنشاء الحملات**: إنشاء روابط تتبع لأي منتج
- **رابط التتبع**: رابط فريد لكل حملة
- **رمز QR**: تحميل QR Code للإعلانات
- **تتبع الأداء**:
  - عدد النقرات
  - التحويلات
  - معدل التحويل
  - الأرباح من العمولات

## 📱 استخدام المنصة

### للبدء:
1. انتقل إلى الصفحة الرئيسية
2. اضغط على "إنشاء حساب"
3. املأ بيانات التسجيل
4. **اختر دورك** (هذه أهم خطوة):
   - عميل/مشتري
   - مصمم/مبدع
   - مصنع/مزود خدمة
   - مسوق/وكالة

### رحلة العميل:
1. انتقل إلى **المحفظة**
2. اضغط **"+ شحن المحفظة"**
3. اختر **تمويل BNPL** وأدخل المبلغ (مثال: 1000 ريال)
4. شاهد الرصيد المقسم (700 حر + 300 مقفل)
5. ارجع إلى **المول** واختر منتج
6. اضغط **"اشتر الآن"** ← **"تأكيد الدفع"**
7. لاحظ الخصم من الرصيد المقفل أولاً!

### رحلة المصمم:
1. اضغط **"+ رفع تصميم جديد"**
2. ارفع الملف وأدخل التفاصيل
3. اضغط **"توثيق وحفظ الحقوق"**
4. اختر **"البيع كمنتج ملموس"**
5. اختر المنتج الخام والمصنع
6. حدد ربحك وشاهد السعر النهائي
7. اضغط **"نشر في المول"**

### رحلة المصنع:
1. انتقل إلى تبويب **"كتالوجي"**
2. اضغط **"+ إضافة منتج خام جديد"**
3. املأ التفاصيل والتكلفة
4. اضغط **"حفظ المنتج"**
5. عند وصول طلب جديد، افتحه من تبويب **"الطلبات"**
6. حمل ملف التصميم واطبع بوليصة الشحن
7. اضغط **"تأكيد الشحن"** لتوزيع الأرباح

### رحلة المسوق:
1. اضغط **"+ إنشاء حملة جديدة"**
2. أدخل اسم الحملة ورابط المنتج
3. انسخ **رابط التتبع** أو حمل **QR Code**
4. شارك الرابط مع جمهورك
5. تابع الأداء في لوحة التحكم

## 🎯 الميزات الرئيسية

### نظام المحفظة (تفصيل كاش)
- **الرصيد الحر**: متاح للاستخدام في أي مكان
- **الرصيد المقفل**: يُستخدم أولاً في المشتريات داخل المنصة
- **تمويل BNPL**: نظام 70/30 بدون فوائد

### توثيق NFT
- توثيق تلقائي للتصاميم على Polygon
- حماية الحقوق الفكرية
- إثبات الملكية

### نظام التسعير
```
السعر النهائي = تكلفة المصنع + عمولة المنصة (30%) + ربح المصمم
```

### نظام التسويق بالعمولة
- روابط تتبع فريدة لكل حملة
- QR Codes قابلة للتحميل
- تتبع النقرات والتحويلات
- عمولات تلقائية
