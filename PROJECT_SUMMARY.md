## 🎯 الشاشات الرئيسية (Aha Moments)

### 1. شاشة اختيار الدور
**المسار**: `/select-role`
- أهم شاشة في المنصة كلها
- 4 خيارات واضحة مع أيقونات
- تحدد مسار المستخدم بالكامل

### 2. شاشة المحفظة (BNPL)
**المسار**: `/app/wallet`
- نظام 70/30 واضح ومرئي
- الرصيد الحر 🔓 + الرصيد المقفل 🔒
- خيارات الشحن (عادي / BNPL)

### 3. شاشة الدفع
**المسار**: `/app/checkout/[id]`
- توضيح استخدام الرصيد المقفل أولاً
- عرض التقسيم بين الأرصدة
- ملاحظة واضحة للمستخدم

### 4. شاشة رفع التصميم
**المسار**: `/designer/upload`
- رفع الملف سهل
- ملاحظة توثيق NFT واضحة
- UX ممتاز

### 5. شاشة الاختيار المزدوج
**المسار**: `/designer/select-mode/[id]`
- خياران واضحان (Store vs Auction)
- شرح مميزات كل خيار
- تصميم جذاب

### 6. شاشة إعداد المنتج
**المسار**: `/designer/setup-product/[id]`
- نظام تسعير شفاف
- حساب تلقائي للسعر النهائي
- عرض واضح للأرباح

### 7. شاشة إضافة منتج خام
**المسار**: `/factory/add-product`
- نموذج بسيط وواضح
- حقل التكلفة الأساسية مهم
- هذه التكلفة يراها المصممون

### 8. شاشة تنفيذ الطلب
**المسار**: `/factory/order/[id]`
- جميع أدوات المصنع في مكان واحد
- تحميل التصميم + طباعة البوليصة
- تأكيد الشحن → توزيع الأرباح

### 9. شاشة نجاح الحملة
**المسار**: `/marketer/campaign/[id]`
- رابط التتبع مع زر نسخ
- QR Code قابل للتحميل
- إحصائيات واضحة
