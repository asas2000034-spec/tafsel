## 🎯 اختبار الرحلات الأربعة

### 1️⃣ رحلة العميل (5 دقائق)

```
1. إنشاء حساب → اختر "عميل"
2. المحفظة → شحن المحفظة
3. اختر "BNPL" → أدخل 1000
4. شاهد: 700 حر + 300 مقفل ✨
5. المول → اختر منتج
6. اشتر → لاحظ الخصم من المقفل أولاً!
```

**النقطة الأساسية:** الرصيد المقفل يُستخدم أولاً 🔒

---

### 2️⃣ رحلة المصمم (5 دقائق)

```
1. إنشاء حساب → اختر "مصمم"
2. رفع تصميم جديد
3. املأ البيانات → "توثيق وحفظ الحقوق"
4. اختر "البيع كمنتج ملموس"
5. اختر منتج خام + حدد ربحك
6. نشر في المول ✓
```

**النقطة الأساسية:** توثيق NFT تلقائي لحماية الحقوق 🎨

---

### 3️⃣ رحلة المصنع (5 دقائق)

```
1. إنشاء حساب → اختر "مصنع"
2. كتالوجي → إضافة منتج خام
3. املأ التفاصيل + التكلفة (50 TFT)
4. حفظ المنتج
5. الطلبات → افتح طلب
6. تأكيد الشحن → توزيع الأرباح ✓
```

**النقطة الأساسية:** توزيع أرباح تلقائي عند الشحن 🏭

---

### 4️⃣ رحلة المسوق (3 دقائق)

```
1. إنشاء حساب → اختر "مسوق"
2. إنشاء حملة جديدة
3. أدخل اسم الحملة + رابط المنتج
4. انسخ رابط التتبع
5. شارك → احصل على عمولات ✓
```

**النقطة الأساسية:** روابط تتبع + QR Codes للعمولات 📢

## 🔥 الميزات الأساسية

### نظام BNPL (70/30)
- ✅ 0% فائدة
- ✅ تقسيم ذكي: 70% حر + 30% مقفل
- ✅ الرصيد المقفل يُستخدم أولاً

### توثيق NFT
- ✅ تلقائي عند رفع التصميم
- ✅ على شبكة Polygon
- ✅ حماية الحقوق الفكرية

### التسعير الشفاف
```
السعر = المصنع + المنصة (30%) + المصمم
```

### التسويق بالعمولة
- ✅ رابط تتبع فريد
- ✅ QR Code قابل للتحميل
- ✅ عمولات تلقائية
